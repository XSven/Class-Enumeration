=head1 NAME

Class::Enumeration::Builder - Builder module for enum classes

=head1 SYNOPSIS

  use strict;
  use warnings;

  package TrafficSignal;

  # Pre-declare the as_string() method
  use subs 'as_string';

  use Class::Enumeration::Builder (
    GREEN  => { action => 'go' },
    ORANGE => { action => 'slow down' },
    RED    => { action => 'stop' }
  );

  sub action {
    my ( $self ) = @_;

    $self->{ action }
  }

  sub as_string {
    my ( $self ) = @_;

    $self->action . ' if ' . $self->name
  }

  1

=head1 DESCRIPTION

Loading C<Class::Enumration::Builder> starts the build process. It makes
each enum class inherit from the abstract parent class L<Class::Enumeration>.

If you want to override the default stringification you have to pre-declare
the C<as_string()> before loading the builder.
